This code calculates the multiplication factor and the the neutron flux in a slab of 1 m thickness.
The code is written in C++ and ploting the flux using matplotlib
The code runs 1000 cycles each cycle with 10000 particles.
The user can modify the number of particles or cycles by changing the N_neutrons and N_cycle in CP1.cc file.
##############################################################################################################
 How to run :
 download the file CP1_kholodmahmoud
 in the terminal run the shell script run.sh after you run 'chmod +x run.sh'  
 The multiplication factor will pring in the terminal and the graph of the neutron flux will appear on the screen. 
 
 Open html/index.html for the doxygen of the code documentation.
